      ## How to use images in mdx file for a specific page


        >
        > 1. Images come from './src/images/about', where we add these for content in a page about us
        >
        > 2. These images we can add directly in a './src/markdown/pages/about.mdx'
        >
        > 3. For ex.: 'props.images.Photo1Fixed ' or  'props.images.Photo1FLuid'
        >
        > 4. The image name is divide into two pieces: 'Photo1' and 'Fluid' or 'Fixed'
        >
        > 5. The first part is a natural name of file inside './src/images/about', the second part of the name is a
        part of graphql childrenImageSharp fragment
        >
        > 6. 'Photo1Fixed' props: align, height and width
        > 6. 'Photo1Fluid' without props.
